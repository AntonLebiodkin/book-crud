# book-crud
Test task for Junior JS position

### To start project
```
    git clone https://github.com/AntonLebiodkin/book-crud/
    cd book-crud
    npm install
    npm start
```

### Prerequisities

All dependencies given in package.json file

```
    "body-parser": "^1.15.2",
    "express": "^4.14.0",
    "jade": "^1.11.0",
    "mongodb": "^2.2.5",
    "mongoose": "^4.5.7"
```
